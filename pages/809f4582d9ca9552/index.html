# 兄弟组件传值

子组件1中把值传到父组件,父组件获取值传入子组件2

父组件：

<子组件1 @方法名x="方法名y"></子组件1>

<子组件2 :值名称x="值x"></子组件2 >

data() {
	return {
	 值x: ''
	}
},
methods: {
	方法名y(值) {
		this.值x = 值
	}
}
